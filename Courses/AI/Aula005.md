# AI ethics

## Ethical pillars
```
Fairness - Modelo justo e imparcial

Robustness - O modelo faz o que é suposto sem
consequencias negativas 

Explainability - Facilidade ao explicar

Transparency - Saber o que o modelo faz e como 
(como os componentes de nutrientes de um alimento)

Privacy - Respeita?
```

# Transparencia:

```
O sistema que usa certos dados tem que 
	1) Dizer como teve os dados e de quem é qu e são os dados 
	2) Quem tem acesso aos dados usados

transparency:
is disclosing information related to the data used for 
building AI systems, design decisions made throughout the process, 
model creation, model evaluation, and model deployment.

Conceito: AI MODEL LEAD
```
